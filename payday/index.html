Payday 2 tommy modding guide

Prerequisite

If you've previously used vanilla BLT, you will have to:

Delete IPHLPAPI.dll - This is the original BLT DLL, and you can't have both installed at the same time due to conflicts.

Delete mods/base - This is another part of the original BLT, and SuperBLT will download its own version. However, you must first delete the old version.

Step 1 - Install SuperBLT

Install SuperBLT from: https://sblt-update.znix.xyz/pd2update/download/get.php?src=homepage&id=payday2bltwsockdll

Drag the dll into the payday 2 folder in steam

OPEN PAYDAY 2 ONCE, it will then create the mods folder in payday 2.

Step 2 - Install mods

Click the links to be sent to the modworkshop link, press the big download button.

Open the zip file and drag the folder into the Payday 2 mods folder.

Essentials

  • The Fixes, has many different bug fixes and crash fixes.
  • Visual Fixes, contains visual bug fixes.
  • BeardLib, adds auto updating to mods.
  • Vanilla HUD Plus, is the most lightweight custom HUD, is very close to vanilla but is a lot less dogshit, contains a lot of useful counters etc.

QoL fixes

  • Little Intelligence EnhancementS, updates the aging Payday 2 2 AI.
  • Meth Helper Updated, helps cook meth on rats, cook off, lab rats and border crystals
