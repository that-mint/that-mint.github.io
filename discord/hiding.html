Hide blocked users from Discord.

  1. Close Discord

  2. Download BetterDiscord and install it

  3. Install the RemoveBlockedUsers plugin. (Settings > BetterDiscord > Plugins > Open Plugins Folder, Drag the .js file into here)

  4. Reload and enable the plugin.

  5. Block the person you don't want to see anymore.

  6. If it stops working in future, it can usually  be fixed by going to BetterDiscord, redownloading it and choosing "Repair" instead of install.
